# PROJETO DE SISTEMA BANCÁRIO 2.0

Este projeto tem por finalidade a modularização da estrutura de funcionamento de uma conta bancária, com depósitos, saques, extratos de conta e a criação de novas funções como cadastro de clientes e contas associadas a cada cliente. Além de explorar os fundamentos básicos da linguagem Python, estruturas condicionais e listas, este projeto aborda a criação de funções otimizando a leitura do código.

Esta é a segunda etapa de um projeto maior em desenvolvimento do curso de Python.
